# pack

`pack` is a Rust research codec for lossless recursive compression.

The project explores a shared formal base where a compact key `K` describes an
executable block generator and a correction vector `V` restores the exact input.

## Current architecture

```text
input block
  -> topology analysis
  -> executable K (maximum 512 bits)
  -> exact self-inverse Program + Feistel + U_K path
  -> shared terminal + parity trajectory V
  -> strict K + V + overhead budget
  -> recursive archive layers
```

The implementation also contains trajectory, alphabet, and raw strategies. The
smallest valid block representation is selected.

## Build and test

```bash
make check
make test
make release
```

Direct Cargo commands are also supported:

```bash
cargo fmt -- --check
cargo check --all-targets
cargo test
```

## CLI

```bash
cargo run --release -- pack <input> <output.pack>
cargo run --release -- unpack <output.pack> <restored>
cargo run --release -- benchmark tests/noise
```

## Measured status

All measurements use external files and verify byte-for-byte roundtrip.

| Corpus | Result |
|---|---:|
| 15 files under `tests/noise` | `29,722 -> 26,158` bytes each (`1.1362x`) |
| `tests/test1.txt` | `24,465 -> 10,963` bytes (`2.2316x`) |
| `tests/test2.txt` | `22,673 -> 10,178` bytes (`2.2276x`) |

The research target of `>=10x` on the external noise corpus is not achieved.
Synthetic quality fixtures generated by the codec itself are intentionally not
used.
