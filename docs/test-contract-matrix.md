# `pack` Contract Test Matrix

**Status:** 64 tests.
**Command:** `cargo test`

This matrix describes executable contracts. It does not treat data generated by
the codec itself as an external compression-quality corpus.

## Core contracts

| Area | Contract | Tests |
|---|---|---|

## Removed misleading contracts

The former tests that generated input with the codec's own predictor or
trajectory helper and then asserted `>=10x` compression were removed. They
proved only that a decoder can compactly describe its own output, not that the
metagenerator works on an external corpus.

Compression quality is measured only through the public CLI against files under
`tests/`.
